Comment citer ces sources dans ton rapport :
Pour chaque donnée, la chaîne de traçabilité est :
Rapport annuel officiel (ou communiqué de résultats) → feuille Financials_Raw du classeur Excel → Java ETL → processed_bank_data.csv → Power BI

Exemple de citation : « Résultat net BNP Paribas 2024 : 11 688 M€ — Source : BNP Paribas, Communiqué de résultats annuels 2024, publié le 5 février 2025, invest.bnpparibas/resultats »
